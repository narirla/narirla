## 👋 데이터 분석가 김나리입니다

데이터를 기반으로 문제의 원인을 분석하고  
패턴을 도출하여 실제 활용 가능한 인사이트로 연결하는 것을 목표로 합니다.  

단순한 결과 해석이 아닌, 데이터의 분포와 전처리 과정까지 확인하며  
분석의 신뢰도를 높이는 것을 중요하게 생각합니다.

- **Language** : Python, SQL, Java
